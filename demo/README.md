# Heka Identity Platform Demos

This folder contains a collection of demo applications / setups showcasing relevant and promising decentralized identity use cases implemented with the Heka Identity Platform.
Each demo provides a practical example of how to leverage the platform's components and supported standards to solve real-world identity challenges.

Please note that the provided demo code is not intended for production usage (see `Disclaimer` section in specific demo READMEs, where present).

## Available Demos

- [Agent-to-Agent (A2A) + OID4VP integration](./a2a-oid4vp): Demonstrates how an AI agent can request additional authentication from a user using the OID4VP protocol, leverages Heka Identity Platform as Verifiable Credentials Wallet and Verifier.
